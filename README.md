# Task Manager CLI (CS65 Coursework)

## Overview
This project is a Python command-line task management system developed for the CS65 Software Development coursework.

The application allows users to:
- Add new tasks
- View existing tasks
- Mark tasks as completed
- Delete tasks

Tasks are stored persistently using a JSON file.

## How to Run
1. Activate virtual environment:
   .\.venv\Scripts\Activate.ps1
2. Run the program:
   python src/main.py

## Features
- Menu-driven CLI
- Persistent storage using JSON
- Error handling for invalid input
- Modular code structure

## Data Storage
Tasks are saved in `data/tasks.json`.
